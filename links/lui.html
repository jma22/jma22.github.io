I joined a team in the Media Lab to help develop voice control and AR functions for their multimodal website.

Large User Interface

LUI is a gesture and voice controlled user interface, aimed to facilitate controls on large user interfaces without a mouse and keyboard. This linklonk probably explains it better.

What I did

I joined when most of the UI and gesture controls were created. So I had to design a backend with Google Action,Firebase and DialogFlow that can work with the React-based gesture controls.

I created basic commands to navigate between pages, but also multimodal commands such as 'openthis', to open a file a user is currently pointing at. I also played around with the Prismatic API for creating AR content on a website.
