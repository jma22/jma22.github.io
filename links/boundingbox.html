Problem:

Bounding Box, given the points of a polygon. Can we figure out if another point is inside it?

Yes, I know this is an easy problem to solve. linklonk But this problem has a few great pros:

  • It is easy to create a lot of data for this problem.
  • It is easy to solve with heuristic -- so it probably is also easily solvable with ML.
  • Great for familiarizing with pytorch!
